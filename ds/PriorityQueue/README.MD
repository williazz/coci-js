# Priority Queue

A priority queue is an abstract data type: it collects a set a values and retrieves them according to a specified priority. Higher priority elements are served before lower ones.

| Operation  | Avg Time |
| ---------- | :------: |
| getRoot    | constant |
| deleteRoot |  log(n)  |
| insert     |  log(n)  |
| heapify    |  linear  |
